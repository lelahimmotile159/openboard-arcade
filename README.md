OpenBoard Arcade offers a collection of digital board games and minigames. You can play titles like Ludo and Chess on your Windows computer. Follow these steps to set up the game on your workstation:

3. Windows might display a security prompt. If you see a blue window saying "Windows protected your PC," click "More info" and then click "Run anyway."
4. Follow the instructions on the screen.
5. Choose your preferred installation folder or accept the default location.
6. Click Install to start the process.
7. Wait for the progress bar to finish.
8. Click Finish to close the installer.

Once you finish the installation, you can start the application:

1. Locate the OpenBoard Arcade icon on your desktop.
2. Double-click the icon to open the main menu.
3. Use your mouse to navigate through the game library.
4. Select a game to view the rules.
5. Click Play to start your session.

## 🛠 Troubleshooting Common Issues

If the software does not behave as expected, look at these common solutions to fix your issues.

### The Screen Flickers or Shows Black
This usually happens because your graphics drivers are old. Visit the website of your computer manufacturer, such as Dell, HP, or Lenovo, to download the latest video drivers. Restart your computer after you update the drivers.

### The Game Reaches a Slow State
If the action feels laggy, turn off other programs running in the background. Web-based games like this use your computer memory to load graphics. Closing your web browser or other office applications frees up memory for the game.

### The Game Files Fail to Open
Sometimes security software blocks new game files. If your antivirus warns you about the file, check your antivirus settings and mark the OpenBoard Arcade folder as a trusted location. 

### How to Update
When a new version becomes available, you simply download the new installer from the link above and run it again. The installer replaces the old files while keeping your game settings intact.
